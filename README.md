## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Clone

```bash
$ git clone --recursive https://github.com/nico989/SVMN.git
```

### Working Directory

```bash
$ cd SVMN
```

### Permissions

```bash
$ chmod -R +x scripts
```

### Initialize

```bash
$ scripts/init.sh
```
